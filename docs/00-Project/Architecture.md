# Architecture

Ambience TutorsFlow™ is a SaaS tutoring platform built with a modern web architecture.

## System Flow

User → React Frontend → Supabase Client / Express Backend → PostgreSQL Database

## Core Components

- React frontend for user interface
- Node.js/Express backend for secure server operations
- Supabase for authentication, PostgreSQL, Row Level Security, and API access
- PostgreSQL database for persistent data
- Future AI services for question generation, lesson planning, and step-by-step solutions

## User Roles

- Platform Admin
- Organization Admin
- Tutor
- Parent
- Student

## Core Data

- Organizations
- Tutors
- Parents
- Students
- Bookings
- Assignments
- Messages
- Invoices
- IEP records
- Character education records
