## 🚀 Key Implementation Pillars

- **Visual Excellence**: Dark-theme aesthetics utilizing curated HSL color parameters, subtle micro-animations (`scaleUp`, `fadeIn`, `spinSlow`), and elegant glassmorphic backdrops.
- **Universal Utilities**: Global search mapping across 9 core database entities and a categorized, interactive notification badge system integrated globally within the navigation header.
- **Extreme Portability**: Perfect responsive viewports on desktop, tablet, and mobile with strict adherence to the browser boundaries (`max-width: 100%` within the centered `1126px` container).
- **Accessibility Compliance**: Semantic HTML tags, clear `aria-label` elements, active focus outline styles, and keypress Esc-listeners.
