## [1.15.0] — 2026-06-27 (Phase 15 Completed)
### Added
- **Universal Database Search Modal**: Engineered an ultra-fast client-side federated query engine matching across 9 distinct core entity tables in real-time (Students, Tutors, Parents, Assignments, Homework worksheets, Lesson plans, Invoices, Messages, and Reports).
- **Interactive Notification Center**: Added a header bell dropdown with dynamic unread badge indicators and 8 stylized message classifications (homework reminders, slots, message pings, study updates, subscription renewals, invoice confirmations, progress charts, and virtue achievements).
- **Accessibility ARIA Upgrades**: Outfitted all interactive toggles with screen-reader labels, semantic markups, and active focus boundaries (`*:focus-visible` outlines), plus custom Escape-key keydown handlers.
- **Mobile Responsiveness Audits**: Perfected touch-height target ranges (`44px` minimum) and grid column wrappers for flawless execution on phone, tablet, and desktop viewports.
- **100% Production Build Integrity**: Verified error-free Vite front-end compilation (under 561ms) and Node route parsing syntax checks with exit code 0.

## [1.14.0] — 2026-06-27 (Phase 14 Completed)
- **Redesigned Public Pricing Page**: Structured into three distinct domains: **Student AI Plans**, **Professional Tutor Plans** (Starter, Flex, Professional, Elite), and **Institutional Plans** with an interactive 20% discount yearly billing toggle and extensive feature matrix.
- **Rebranded "My Plan" Customer Page**: Consolidated user-facing subscriptions into an active quota hub, displaying live visual limits trackers (worksheets, AI request tokens, tutor hours), billing details (renewal date, edit card inputs), promotional coupon field, and printable receipt tables.
- **Flagship Multi-Format AI Homework Workspace**: Expanded homework assistant to support math/science worksheets, PDF files, images of handwritten notes, and coding assignments, utilizing the complete 9-step Socratic teaching scaffold.
- **Premium Glassmorphic Upgrade Interstitials**: Placed protective glowing warning cards on locked premium actions for Free tier users, linking directly to plan upgrades.
- **Intelligent AI Study Vault**: Subject-categorized (Math, Science, English, etc.) revision vault allowing students to search historical logs, filter by type, check concept mastery progress, and slide-open detailed Socratic walkthroughs.
- **SaaS Telemetry Command Cockpit**: Deployed an executive admin panel tracking key financial SaaS performance indicators: Monthly Recurring Revenue (MRR: $14,850), Annual Recurring Revenue (ARR: $178,200), user growth ratios, subscriber churn (1.8%), and renewal success (98.2%).
- **Stripe SSL Sandbox Integration**: Built sandbox coupon processor (`SOLIDEOGLORIA` 50% discount), secure mock token handlers, and automated customer portal triggers.

### Changed
- Refined dashboard navigation, replacing "Subscriptions" tabs with intuitive "My Plan" labels for students, parents, and tutors.
- Supercharged Admin Dashboard with live platform performance charts.
