## 3. Financial Billing Operations
* **Discount System**: Users subscribing to Yearly billing automatically receive a **20% discount** applied at checkout, calculated as: `discounted_price = Math.floor(monthly_price * 0.8)`.
* **Stripe Webhook Sync**: Upgrade actions utilize server-to-server POST triggers synced with local database sandboxes and Supabase triggers.
* **Refund Guard**: Securely backed by our *100% Grace-Period Satisfaction Guarantee*.

---

## 4. Phase 13 Enhancements (My Plan Rebranding & Billing Sandbox)
* **"My Plan" Rebranding**: Renamed the user-facing "Subscriptions" tab to "My Plan" across Student, Parent, and Tutor dashboards. This aligns dashboard navigation with intuitive self-service terminology.
* **Administrative Scope Retention**: Retained the administrative "Subscriptions" label on the Admin Dashboard for clear directory control of all tenant licenses.
* **Unified Workspace Experience**: Integrated the pricing cards, current tier summary, usage statistics, card updater, and past invoices table into a premium cohesive dashboard panel.
* **Simulated Billing Sandbox**: Implemented an interactive credit card updater and receipts ledger. Users can print beautiful browser receipts and download textual records.
* **Elegant Tab Navigation**: Solved tab overflow globally using smooth horizontal swipe scrolling with slim native scrollbars, preventing layout breakage across all screen widths.

## 5. Phase 15 Enhancements (Universal Search & Notification Center)
* **Universal Database Search**: A federated, client-side database search engine tracking 9 distinct core tables in real-time, available standard across all billing tiers (Free to Enterprise) to optimize search discovery.
* **Interactive Notification Center**: A navigation header-mounted alert bell dropdown tracking 8 event categories with dynamic category-specific colors and toggle controls, provided standard to all user profiles to enhance platform UX.
* **Accessibility & Mobile Audits**: Accessible semantic HTML structures, keyboard escape listeners, and focus indications (`*:focus-visible`) integrated globally across all subscriber platforms.
