## 🚀 Professional Resume Bullet Points

* **Full-Stack SaaS Platform Development**: Architected and launched **Ambience TutorsFlow™**, a multi-tenant subscription SaaS learning platform combining pedagogical AI, video scheduling, and financial transaction controllers.
* **Dual-Mode Portability Orchestration**: Engineered a zero-configuration dual-mode execution strategy, allowing the React 19 app and Node Express server to automatically fall back to local offline memory states (`mockData.js`) if cloud credentials are absent, streamlining evaluation and offline deployment.
* **Security & Custom Rate-Limiting**: Built an Express.js sliding-window memory rate limiter (capped at 15 req/min) using a Javascript `Map` registry to secure Stripe payments and Zoom endpoints, reducing infrastructure dependencies by eliminating external database requirements.
* **Zoom OAuth & Scheduling Automation**: Designed a background OAuth token refresher in Node.js, caching Zoom bearer keys and refreshing credentials dynamically to ensure uninterrupted video meeting allocations on tutor bookings.
* **Strategy-Pattern Transaction Settlers**: Implemented a Strategy-Pattern payment engine supporting Stripe Checkout, PayPal integration, and manual Zelle reconciliations, including automated HTML invoice rendering and browser-printable billing receipts.
* **Pedagogical AI Integration**: Formulated generative prompts using Google Gemini 2.5 Flash to power special-needs IEP diagnostics, character-aligned lesson planners (Grit, Perseverance), and real-time subject copilots across 11 disciplines.
* **Vanilla CSS Glassmorphic Styling**: Developed a premium responsive dark theme using Vanilla CSS HSL tokens, reducing CSS payloads and achieving smooth rendering across phone, tablet, and desktop viewports.
* **ADA & Accessibility Compliance**: Achieved strict ADA accessibility compliance by integrating semantic HTML5 structures, keyboard-escape event listeners, and tabbing focus outlines (`*:focus-visible`) restricted to keyboard interactions.
* **Enterprise Analytics Telemetry**: Built an administrative SaaS Telemetry dashboard, aggregating active MRR ($14,850.00), ARR ($178,200.00), user retention ratios, subscription churn metrics, and system audit logs in real-time.
