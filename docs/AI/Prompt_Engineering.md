# Prompt Engineering

## Purpose

Prompt engineering guides the AI to produce structured, safe, educational, and student-friendly responses.

## Core Prompt Requirements

- Explain concepts clearly
- Provide hints before full answers
- Show step-by-step reasoning
- Include common mistakes
- Generate similar practice
- Adapt to grade level
- Avoid simply completing homework without teaching

## Output Format

AI responses should include:

1. Concept overview
2. Guided hint
3. Step-by-step walkthrough
4. Final answer
5. Common mistake
6. Similar practice question
7. Teacher or parent note when needed
