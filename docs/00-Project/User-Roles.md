# User Roles

## Student
Uses AI tools, homework assistant, practice tests, Study Vault, progress tracking, and learning support.

## Parent
Monitors student progress, receives summaries, communicates with tutors, manages billing, and supports at-home learning.

## Tutor
Manages students, lesson plans, tutoring sessions, AI Copilot tools, IEP support, assignments, and reports.

## Administrator
Manages platform operations, analytics, subscriptions, revenue, users, schools, and intelligence dashboards.

## School / Institution
Uses organization-level reporting, student analytics, tutor/teacher oversight, and institutional subscription tools.
